Uma breve apresentação:

  • Sketch é um framework que facilita o design de páginas responsívas com HTML e CSS.

  • As regras do Sketch são simples, o que torna fácil o seu aprendizado.

  • Você pode criar layouts criativos mesclando as regras do framework com regras de formatação escritas em arquivos externos.

  • Você pode ainda sobreescrever as classes do framework para adicionar cores e imagens de fundo, ou até mesmo aplicar um novo estilo criativo.

  • Sketch é leve!

    • Apenas dois arquivos CSS: o arquivo principal possui apenas 43 Kb, enquando que o arquivo secundário possuí 155 Kb

    • O arquivo principal do Sketch contém todas as regras necessárias ao design responsívo.

    • O arquivo secundário do Sketch contém regras adicionais para incorporar cores às páginas Web.
